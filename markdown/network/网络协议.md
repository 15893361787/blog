#### 一些概念
+ 只要是在网络上跑的包，都是完整的。可以有下层没上层，绝对不可能有上层没下层。
+ 对 TCP 协议来说，三次握手也好，重试也好，只要想发出去包，就要有 IP 层和 MAC 层，不然是发不出去的。
+ 查看 IP 地址,在 Windows 上是 ipconfig，在 Linux 上是ifconfig。
+ IP 地址是一个网卡在网络世界的通讯地址，相当于我们现实世界的门牌号。
